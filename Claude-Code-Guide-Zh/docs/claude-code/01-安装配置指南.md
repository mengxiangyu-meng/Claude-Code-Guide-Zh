## 1. 什么是 Claude Code

Claude Code 是 Anthropic 官方推出的 AI 编程 CLI 工具，旨在帮助开发者：

- 🤖 **智能编码** - 编写、调试、重构代码
- 📖 **代码理解** - 解读复杂代码库
- 🛠️ **自动化** - 自动化重复性开发任务
- 📝 **文档生成** - 自动生成注释和文档

> 💡 **一句话理解**：Claude Code = 终端里的编程助手，帮你写代码、调 Bug、做架构。

### Claude Code vs 其他 AI 编程工具

| 特性 | Claude Code | GitHub Copilot | Cursor |
|-----|-------------|----------------|--------|
| 开发商 | Anthropic | Microsoft/OpenAI | Anysphere |
| 交互方式 | CLI | IDE 插件 | IDE + CLI |
| 上下文理解 | 超强 | 中等 | 强 |
| 自定义能力 | Hooks/Skills | 有限 | 中等 |
| 价格 | $20/月 | $10/月 | $20/月 |

## 2. 环境要求

### 2.1 操作系统

Claude Code 支持以下操作系统：

- ✅ **macOS** 12.0+ (Monterey 及以上)
- ✅ **Linux** (Ubuntu 20.04+, Debian 10+, Fedora 35+)
- ✅ **Windows** 10+ (通过 WSL2 或 PowerShell)

### 2.2 硬件要求

| 组件 | 最低配置 | 推荐配置 |
|-----|---------|---------|
| 内存 | 4GB | 8GB+ |
| 磁盘 | 1GB | 5GB+ |
| 网络 | 稳定上网 (API 调用) | 稳定上网 |

### 2.3 API Key

Claude Code 需要 Anthropic API Key：

- 访问 [Anthropic Console](https://console.anthropic.com/)
- 注册账号并获取 API Key
- 或使用第三方中转服务（国内用户）

> ⚠️ **注意**：Claude Code 已切换到原生安装，**不再需要 Node.js**！

## 3. 安装步骤

### 3.1 macOS / Linux 安装

```bash
# 方法一：官方一键安装（推荐）
curl -s https://api.anthropic.com | bash

# 方法二：指定版本安装
curl -s https://api.anthropic.com | bash -s -- -v 2.1.52
```

### 3.2 Windows 安装

#### 方式一：PowerShell（推荐）

```powershell
# 以管理员身份运行 PowerShell
irm https://api.anthropic.com | iex
```

#### 方式二：WSL2（适用于需要 Linux 环境的用户）

```bash
# 1. 启用 WSL2
wsl --install

# 2. 进入 Ubuntu
wsl -d Ubuntu

# 3. 安装 Claude Code
curl -s https://api.anthropic.com | bash
```

### 3.3 验证安装

```bash
# 查看版本
claude --version

# 查看帮助
claude --help
```

## 4. API 配置

### 4.1 环境变量配置

#### macOS / Linux

```bash
# 临时生效（当前终端）
export ANTHROPIC_API_KEY="sk-ant-api03-xxxxx"

# 永久生效（写入 ~/.bashrc 或 ~/.zshrc）
echo 'export ANTHROPIC_API_KEY="sk-ant-api03-xxxxx"' >> ~/.zshrc
source ~/.zshrc
```

#### Windows PowerShell

```powershell
# 临时生效
$env:ANTHROPIC_API_KEY = "sk-ant-api03-xxxxx"

# 永久生效
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-api03-xxxxx", "User")
```

### 4.2 使用配置文件

Claude Code 配置文件位于 `~/.claude/settings.json`：

```json
{
  "apiKey": "sk-ant-api03-xxxxx",
  "model": "claude-sonnet-4-20250514",
  "maxTokens": 4096
}
```

### 4.3 API 中转站配置（国内用户）

如果在国内使用官方 API 困难，可以配置中转：

```bash
# 使用 OpenAI 兼容的中转服务
export ANTHROPIC_BASE_URL="https://api.example.com/v1"
export ANTHROPIC_API_KEY="your-key"
```

> 📝 **编辑注**：实际使用时请替换为可用的中转服务地址。

## 5. IDE 集成

Claude Code 可以与多种 IDE 集成，获得更好的编程体验。

### 5.1 VS Code

#### 方式一：Official Claude Code Extension

```bash
# 在 VS Code 中安装扩展
# 1. 打开 VS Code
# 2. 按 Cmd+Shift+P / Ctrl+Shift+P
# 3. 输入 "Extensions: Install Extensions"
# 4. 搜索 "Claude Code" 并安装
```

#### 方式二：VS Code Copilot 插件（替代方案）

```bash
# 安装 VSCode 插件
code --install-extension github.copilot
```

### 5.2 Cursor

Cursor 是基于 VS Code 的 AI 编程编辑器，深度集成 Claude：

1. 下载 [Cursor](https://cursor.sh/)
2. 安装并登录
3. 在 Settings → Models 中选择 Claude 模型

### 5.3 其他 IDE

| IDE | 集成方式 | 支持情况 |
|-----|---------|---------|
| JetBrains | 插件市场搜索 | ✅ 支持 |
| Vim/Neovim | claude.nvim | ✅ 支持 |
| Emacs | claude.el | ✅ 支持 |

## 7. 常见问题

### Q1: 安装后找不到 `claude` 命令？

**解决方案：**

```bash
# 检查是否安装成功
which claude

# 如果没有，尝试重新安装
curl -s https://api.anthropic.com | bash -s -- -f

# 或手动添加到 PATH
export PATH="$HOME/.claude/bin:$PATH"
```

### Q2: API Key 无效或过期？

**解决方案：**

1. 登录 [Anthropic Console](https://console.anthropic.com/)
2. 检查 API Key 状态
3. 如有必要，生成新的 API Key
4. 更新环境变量

### Q3: 网络连接问题？

**解决方案：**

1. 检查网络连接
2. 使用代理
3. 配置 API 中转站（见 4.3 节）

### Q4: Windows 下安装失败？

**解决方案：**

1. 确保使用管理员权限的 PowerShell
2. 检查执行策略：`Get-ExecutionPolicy`
3. 如需允许脚本：`Set-ExecutionPolicy RemoteSigned`

### Q5: 如何查看详细日志？

```bash
# 启用调试模式
claude --debug "your request"

# 查看日志文件
cat ~/.claude/logs/claude.log
```

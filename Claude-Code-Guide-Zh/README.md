## 🚀 快速开始

```bash
# 1. 安装 Claude Code（macOS/Linux）
curl -s https://api.anthropic.com | bash -s --

# 2. 配置 API Key
export ANTHROPIC_API_KEY="your-api-key"

# 3. 开始使用
claude
```
